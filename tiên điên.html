CSS Selectors

Áp dụng CSS vào các phần khác nhau của HTML

1.Bộ chọn thẻ nhắm mục tiêu vào các phần tử dựa vào tên thẻ HTML của chúng.

  1. 2.Bộ chọn class nhắm mục tiêu vào các phần tử dựa trên giá trị của thuộc tính class.
  2. 3.Bộ chọn id nhắm mục tiêu vào các phần tử dựa trên giá trị của thuộc tính id
  3. 4.Bộ chọn thuộc tính nhằm mục tiêu vào các phần tử dựa trên thuộc tính của chúng và giá trị của thuộc tính.
  4. 5. Bộ chọn chung nhằm mục tiêu tới tất cả các phần tử.
